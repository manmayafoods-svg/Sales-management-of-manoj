Export / Import Data

🗂 Full Backup (JSON)
Stock + Sales + Bookings sabai ek file ma.
📥 Import Stock from Excel
Purano Excel (.xlsx/.xls/.csv) ko stock list yaha upload garnus. Columns yeti milnu parcha: Model, Category, Colour, Reg No, Engine No, Chassis No, Year, Price, Status, Purchase Date, Notes (order/case farak bhaye pani chalcha, kehi column khali rahe ni thik cha).
📊 Import Sales from Excel
Purano sales data (.xlsx/.xls/.csv) yaha import garnus. Template download garera format herna sakinchha. Columns: Sales Date, Nepali Date, Sales Days, Invoice No, Model, Color, Chassis No, Engine No, Registration, Bagmati No, Model Year, Customer Name, Address, Telephone, Sales VAT No, Insurance, Insurance Type, Insurance Name, Namsari.
☁️ Google Sheet Backup (Cloud)
Stock + Sales + Bookings sabai cloud ma backup huncha. Pahilo patak Apps Script Web App URL setup garnu parcha.
☁️ Cloud Sync (Supabase) — automatic, sabai device ma
Yo automatic chalu cha — jaba jaba data save garnus, cloud ma pugcha, ra arko computer/phone ma app kholda automatic ta latest data aaucha. Setup ko lagi kehi garnu pardaina.
☁️ Checking...
🔒 App PIN
App khulda sabailai yo PIN chahincha. PIN change garna purano PIN ra naya PIN halnus.
🔐 Admin Delete PIN
Kunai pani record (stock, sale, booking, valuation, receipt, voucher, purchase) delete garda yo separate PIN chahincha — admin le matra thaha huncha. Pahilo patak delete garda automatic set garna sakincha; yaha bata pani change garna sakincha.
⚡ Real-time Sync (Firebase) — 2+ PC ma instant update
Dui ya badi PC ma ekai samaya kaam garda, ek device ko change arko ma instant (1 second bhitra) dekhinchha. Firebase console bata config copy garera tala paste garnus.
